## 1. Введение в язык C++

### 1.1 История и область применения

**C++** — это мощный язык программирования среднего уровня, созданный **Бьярне Страуструпом** в 1985 году. Он является расширенной версией языка **С** с добавлением принципов объектно-ориентированного программирования (ООП).

C++ используется в:

- Системном программировании
- Разработке компьютерных игр
- Высокопроизводительных приложениях
- Встроенных системах и микроконтроллерах
- Компиляторах и редакторах

### 1.2 Характеристики C++

- **Язык среднего уровня**: предоставляет как высокоуровневые абстракции, так и низкоуровневое управление памятью
- **Статическая типизация**: переменные имеют фиксированный тип при объявлении
- **Компилируемый язык**: код преобразуется в машинный код до выполнения
- **Объектно-ориентированный**: поддерживает классы, наследование, полиморфизм
- **Высокая производительность**: минимальный оверхед, прямое управление памятью

## 2. Первая программа: "Hello, World!"

### 2.1 Базовая структура программы

```cpp
#include <iostream>        // Подключаем библиотеку для ввода-вывода

using namespace std;       // Используем стандартное пространство имён

int main() {               // Главная функция программы
    std::cout << "Hello, World!" << std::endl;  // Вывод текста
    return 0;              // Завершение программы со статусом успеха
}
```

### 2.2 Разбор строк кода

| Элемент                | Назначение                                                                |
| ---------------------- | ------------------------------------------------------------------------- |
| `#include <iostream>`  | Подключает библиотеку для работы с потоками ввода-вывода                  |
| `using namespace std;` | Позволяет использовать объекты из пространства `std` без префикса `std::` |
| `int main()`           | Главная функция программы; всегда выполняется первой                      |
| `std::cout`            | Стандартный поток вывода (консоль)                                        |
| `<<`                   | Оператор вставки данных в поток (направлен слева направо)                 |
| `std::endl`            | Конец строки с автоматическим сбросом буфера                              |
| `return 0;`            | Возврат значения 0 (признак успешного завершения)                         |

### 2.3 Компиляция и запуск

Для компиляции программы используется команда:

```bash
g++ hello.cpp -o hello
```

Для запуска:

```bash
./hello
```

## 3. Переменные и типы данных

### 3.1 Основные типы данных

C++ поддерживает **статическую типизацию** — при объявлении переменной указывается её тип, который не меняется на протяжении существования переменной.

| Тип      | Размер  | Описание                                | Пример                    |
| -------- | ------- | --------------------------------------- | ------------------------- |
| `int`    | 4 байта | Целые числа                             | `int age = 25;`           |
| `float`  | 4 байта | Вещественные числа (одинарная точность) | `float temp = 36.6f;`     |
| `double` | 8 байт  | Вещественные числа (двойная точность)   | `double pi = 3.14159;`    |
| `char`   | 1 байт  | Отдельный символ                        | `char grade = 'A';`       |
| `bool`   | 1 байт  | Логический тип (true/false)             | `bool flag = true;`       |
| `long`   | 8 байт  | Большие целые числа                     | `long bigNum = 1000000L;` |

### 3.2 Объявление и инициализация переменных

```cpp
// Объявление без инициализации
int x;              // x содержит неопределённое значение

// Объявление с инициализацией
int age = 25;       // Инициализация при объявлении

// Инициализация в современном C++ (рекомендуется)
int year{2024};     // Uniform initialization

// Несколько переменных одного типа
int a, b, c;        // Объявления на одной строке
int x = 10, y = 20; // С инициализацией

// Константы
const double PI = 3.14159;  // Переменная не может быть изменена
```

### 3.3 Функция sizeof()

Функция `sizeof()` возвращает размер типа или переменной в байтах:

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "int: " << sizeof(int) << " bytes" << endl;
    cout << "float: " << sizeof(float) << " bytes" << endl;
    cout << "double: " << sizeof(double) << " bytes" << endl;
    cout << "char: " << sizeof(char) << " bytes" << endl;
    cout << "bool: " << sizeof(bool) << " bytes" << endl;

    return 0;
}
```

**Вывод:**

```
int: 4 bytes
float: 4 bytes
double: 8 bytes
char: 1 bytes
bool: 1 bytes
```

## 4. Операторы

### 4.1 Арифметические операторы

```cpp
int a = 10, b = 3;

int sum = a + b;        // Сложение: 13
int diff = a - b;       // Вычитание: 7
int prod = a * b;       // Умножение: 30
int quot = a / b;       // Целочисленное деление: 3
int rem = a % b;        // Остаток: 1

// Инкремент и декремент
a++;                    // a = 11 (постфиксный инкремент)
++a;                    // a = 12 (префиксный инкремент)
a--;                    // a = 11
```

### 4.2 Операторы сравнения

```cpp
int x = 5, y = 10;

bool result;
result = (x == y);      // Равно: false
result = (x != y);      // Не равно: true
result = (x < y);       // Меньше: true
result = (x > y);       // Больше: false
result = (x <= y);      // Меньше или равно: true
result = (x >= y);      // Больше или равно: false
```

### 4.3 Логические операторы

```cpp
bool a = true, b = false;

bool result1 = (a && b);    // И (AND): false
bool result2 = (a || b);    // ИЛИ (OR): true
bool result3 = !a;          // НЕ (NOT): false

// Пример с условиями
int age = 25;
if (age >= 18 && age <= 65) {
    cout << "Вы можете работать" << endl;
}
```

## 6. Условные операторы

### 6.1 Оператор if...else

Используется для выполнения различных блоков кода в зависимости от условия.

```cpp
int age;

if (age >= 18) {

} else {
    cout << "Вы несовершеннолетний" << endl;
}
```

### 6.2 Вложенные условия

```cpp
        cout << "Отлично!" << endl;
    } else {
        cout << "Хорошо!" << endl;
    }
} else {
    cout << "Нужно поучиться" << endl;
}
```

### 6.3 Оператор if...else if...else

```cpp
int grade;
cout << "Введите оценку: ";
cin >> grade;

if (grade == 5) {
    cout << "Отлично!" << endl;
} else if (grade == 4) {
    cout << "Хорошо!" << endl;
} else if (grade == 3) {
    cout << "Удовлетворительно" << endl;
} else if (grade == 2) {
    cout << "Неудовлетворительно" << endl;
} else {
    cout << "Некорректная оценка" << endl;

```

### 6.4 Оператор switch

```cpp
char operation;
double a, b;

cout << "Введите операцию (+, -, *, /): ";
cin >> operation;

cout << "Введите два числа: ";
cin >> a >> b;

switch (operation) {
    case '+':
        cout << "Результат: " << a + b << endl;
        break;
    case '-':
        cout << "Результат: " << a - b << endl;
        break;
    case '*':
        cout << "Результат: " << a * b << endl;
        break;
    case '/':
        if (b != 0) {
            cout << "Результат: " << a / b << endl;
        } else {
            cout << "Ошибка: деление на ноль!" << endl;
        }
        break;
    default:
        cout << "Неизвестная операция" << endl;
}
```

## 7. Циклы

### 7.1 Цикл while

Выполняет блок кода **пока** условие истинно.

```cpp
int i = 1;

while (i <= 5) {
    cout << "Итерация " << i << endl;
    i++;
}
```

**Вывод:**

```
Итерация 1
Итерация 2
Итерация 3
Итерация 4
Итерация 5
```

### 7.2 Цикл do...while

Выполняет блок кода **хотя бы один раз**, потом проверяет условие.

```cpp
int i = 1;

do {
    cout << "Число: " << i << endl;
    i++;
} while (i <= 3);
```

### 7.3 Цикл for

Наиболее гибкий цикл с явной инициализацией, условием и изменением.

```cpp
// Базовый синтаксис: for (инициализация; условие; приращение)
for (int i = 1; i <= 5; i++) {
    cout << "Число: " << i << endl;
}

// Таблица умножения 5×5
cout << "Таблица умножения:" << endl;
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= 5; j++) {
        cout << i << " × " << j << " = " << i * j << "  ";
    }
    cout << endl;
}
```

### 7.4 Операторы break и continue

```cpp
// break: прерывает цикл
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break;  // Выход из цикла
    }
    cout << i << " ";
}
cout << endl;  // Вывод: 1 2 3 4

// continue: пропускает текущую итерацию
for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue;  // Пропускает вывод числа 3
    }
    cout << i << " ";
}
cout << endl;  // Вывод: 1 2 4 5
```

### Задача 6: Проверка простого числа

**Описание:** Проверить, является ли число простым.

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    bool isPrime = true;

    cout << "Введите число: ";
    cin >> n;

    if (n <= 1) {
        isPrime = false;
    } else if (n == 2) {
        isPrime = true;
    } else if (n % 2 == 0) {
        isPrime = false;
    } else {
        // Проверяем делимость на нечётные числа до sqrt(n)
        for (int i = 3; i * i <= n; i += 2) {
            if (n % i == 0) {
                isPrime = false;
                break;
            }
        }
    }

    if (isPrime) {
        cout << n << " — простое число" << endl;
    } else {
        cout << n << " — составное число" << endl;
    }

    return 0;
}
```

**Пример работы:**

```
Введите число: 17
17 — простое число
```

### Задача 7: Обратный порядок цифр

```cpp
    int number;

    return 0;
}
```

### Задача 8: Нахождение максимума и минимума

**Описание:** Прочитать n чисел и найти максимальное и минимальное значения.

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;

    cout << "Сколько чисел вводить? ";
    cin >> n;

    if (n <= 0) {
        cout << "Ошибка: n должно быть положительным числом!" << endl;
        return 1;
    }

    int num, maximum, minimum;

    cout << "Введите число 1: ";
    cin >> maximum;
    minimum = maximum;

    for (int i = 2; i <= n; i++) {
        cout << "Введите число " << i << ": ";
        cin >> num;

        if (num > maximum) maximum = num;
        if (num < minimum) minimum = num;
    }

    cout << "Максимум: " << maximum << endl;
    cout << "Минимум: " << minimum << endl;

    return 0;
}
```

**Пример работы:**

```
Сколько чисел вводить? 5
Введите число 1: 15
Введите число 2: 3
Введите число 3: 42
Введите число 4: 8
Введите число 5: 21
Максимум: 42
Минимум: 3
```

### Задача 9: Конвертация температуры

**Описание:** Перевести температуру из Цельсия в Фаренгейт и наоборот.

```cpp
#include <iostream>
using namespace std;

int main() {
    double temp;
    char unit;

    cout << "Введите температуру: ";
    cin >> temp;

    cout << "Введите единицу измерения (C или F): ";
    cin >> unit;

    if (unit == 'C') {
        double fahrenheit = (temp * 9.0 / 5.0) + 32;
        cout << temp << "°C = " << fahrenheit << "°F" << endl;
    } else if (unit == 'F') {
        double celsius = (temp - 32) * 5.0 / 9.0;
        cout << temp << "°F = " << celsius << "°C" << endl;
    } else {
        cout << "Ошибка: неизвестная единица!" << endl;
    }

    return 0;
}
```

**Пример работы:**

```
Введите температуру: 0
Введите единицу измерения (C или F): C
0°C = 32°F
```
