
### 1. Базовые - объявление классов (50 заданий) - Легкий уровень
Задания 1-50: Создание простых классов с полями различных типов

### 2. Конструкторы и деструкторы (50 заданий) - Средний уровень  
Задания 51-100: Работа с конструкторами (по умолчанию, с параметрами, копирования, перемещения), деструкторами

### 3. Методы класса (50 заданий) - Средний уровень
Задания 101-150: Создание методов для работы с объектами (геттеры, сеттеры, вычислительные методы)

### 4. Инкапсуляция (30 заданий) - Средний уровень
Задания 151-180: Модификаторы доступа, геттеры/сеттеры с валидацией, инкапсуляция данных

### 5. Перегрузка операторов (40 заданий) - Продвинутый уровень
Задания 181-220: Перегрузка арифметических, логических, операторов сравнения, ввода-вывода

### 6. Статические члены (20 заданий) - Средний уровень
Задания 221-240: Статические поля и методы класса

### 7. Указатель this (10 заданий) - Средний уровень
Задания 241-250: Использование указателя this в методах класса

### 8. Дружественные функции (20 заданий) - Продвинутый уровень
Задания 251-270: Friend функции и классы для доступа к приватным членам

### 9. Составные задания (30 заданий) - Продвинутый уровень
Задания 271-300: Комплексные задачи, требующие применения всех навыков

## ПРИМЕРЫ ЗАДАНИЙ ИЗ КАЖДОЙ КАТЕГОРИИ

### Базовые задания (1-50)
1. Создайте класс Point с двумя целочисленными полями x и y
2. Объявите класс Student с полями name (строка) и age (целое число)
3. Создайте класс Rectangle с полями width и height типа double
4. Объявите класс Book с полями title, author и year
5. Создайте класс Circle с приватным полем radius типа double

### Конструкторы и деструкторы (51-100)
51. Добавьте конструктор по умолчанию для класса Point
52. Создайте конструктор с параметрами для класса Rectangle
53. Реализуйте конструктор копирования для класса String
54. Добавьте деструктор для класса Array, освобождающий память
55. Создайте перегруженные конструкторы для класса Time

### Методы класса (101-150)
101. Создайте метод getX() для получения значения x в классе Point
102. Реализуйте метод setRadius() для класса Circle
103. Добавьте метод area() для вычисления площади прямоугольника
104. Создайте константный метод display() для вывода данных объекта
105. Реализуйте статический метод getCount() для подсчета объектов

### Инкапсуляция (151-180)
151. Реализуйте инкапсуляцию для класса BankAccount с приватным balance
152. Создайте публичные геттеры и сеттеры для приватных полей
153. Добавьте защищенные поля для использования в наследниках
154. Реализуйте валидацию в сеттере для поля age (0-120)
155. Создайте приватный метод для внутренних вычислений

### Перегрузка операторов (181-220)
181. Перегрузите оператор + для сложения двух объектов Point
182. Реализуйте оператор - для вычитания комплексных чисел
183. Перегрузите оператор * для умножения вектора на скаляр
184. Реализуйте оператор / для деления дробей
185. Перегрузите оператор == для сравнения двух строк

### Статические члены (221-240)
221. Создайте статическое поле count для подсчета объектов класса
222. Реализуйте статический метод getInstanceCount()
223. Добавьте статическую константу PI в класс Circle
224. Создайте статический метод create() для фабрики объектов
225. Реализуйте статический метод compare() для сравнения объектов

### Указатель this (241-250)
241. Используйте this для разрешения конфликта имен параметра и поля
242. Реализуйте метод, возвращающий *this для цепочки вызовов
243. Добавьте метод compare(), сравнивающий this с другим объектом
244. Создайте метод assign(), принимающий ссылку и проверяющий this != &other
245. Реализуйте перегрузку оператора =, возвращающую *this

### Дружественные функции (251-270)
251. Объявите дружественную функцию для доступа к приватным полям
252. Реализуйте friend функцию для перегрузки оператора +
253. Создайте дружественную функцию swap() для обмена объектов
254. Объявите дружественный класс для доступа к приватным данным
255. Реализуйте friend оператор << для вывода приватных данных

### Составные задания (271-300)
271. Создайте класс Matrix с динамической памятью и всеми необходимыми методами
272. Реализуйте класс String с перегрузкой операторов и управлением памятью
273. Создайте класс Vector с поддержкой итераторов и алгоритмов STL
274. Реализуйте класс LinkedList с методами добавления, удаления и поиска
275. Создайте класс SmartPointer с подсчетом ссылок

## МЕТОДИЧЕСКИЕ РЕКОМЕНДАЦИИ

### Для студентов:
1. Начинайте с базовых заданий и постепенно переходите к сложным
2. Каждое задание реализуйте в отдельном файле
3. Тестируйте код на различных входных данных
4. Добавляйте комментарии к коду
5. Используйте отладчик для понимания работы программы

### Для преподавателей:
1. Задания можно использовать как домашние работы
2. Рекомендуется давать 5-10 заданий за раз
3. Начинайте с легких категорий
4. Постепенно увеличивайте сложность
5. Используйте задания для контрольных и экзаменов

## КРИТЕРИИ ОЦЕНКИ

**Отлично (5):**
- Корректная реализация всех требований
- Правильная работа с памятью
- Хороший стиль кода
- Наличие комментариев
- Обработка ошибок

**Хорошо (4):**
- Корректная реализация основных требований
- Незначительные недочеты в стиле
- Базовая обработка ошибок

**Удовлетворительно (3):**
- Задание выполнено, но с ошибками
- Отсутствует обработка ошибок
- Плохой стиль кода

**Неудовлетворительно (2):**
- Задание не выполнено
- Код не компилируется
- Грубые ошибки
