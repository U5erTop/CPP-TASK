# Экзамен по системному программированию
## Тема: язык C++ и принципы объектно-ориентированного программирования

Этот конспект оформлен в виде 30 экзаменационных билетов. В каждом билете есть 4 теоретических вопроса и 1 практическая задача на поиск ошибки в коде.

3. Структура простой программы на C++.
4. Понятия переменной, типа данных и области видимости.

**Практическая задача — найти ошибку:**
```cpp

using namespace std;

    int a = 10;
    cout << A << endl;

```
Ошибка: используется идентификатор `A`, а объявлена переменная `a`.

## Основные принципы ООП
1. **Инкапсуляция** — объединение данных и методов работы с ними внутри класса, а также сокрытие внутренней реализации.
2. **Наследование** — возможность создавать новый класс на основе существующего.
3. **Полиморфизм** — возможность использовать единый интерфейс для объектов разных типов.
4. **Абстракция** — выделение главных характеристик объекта без лишних деталей реализации.

## Как готовиться по билетам
- Повтори базовый синтаксис C++: типы, операторы, условия, циклы, массивы, функции.
- Отдельно выучи работу с памятью: указатели, ссылки, `new`, `delete`, время жизни объектов.
- Особое внимание удели ООП: классы, конструкторы, деструкторы, наследование, виртуальные функции.
- По каждому практическому заданию умей объяснить не только ошибку, но и способ исправления.
